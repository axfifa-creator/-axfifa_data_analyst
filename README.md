**Анализ данные для агентства недвижимости**

  1.Исследовательский анализ данных рынка недвижимости Санкт-Петербурга и городов Ленинградской области.

**Исследовательский анализ рынка общепита Москвы**

**А/В-тест для развлекательного приложения и анализ результатов**
  — Цель проекта состояла из нескольких частей:   
  1. Загрузить данные и познакомиться с их содержимым.
  2. Подготовка к проведению теста:
    - расчёт размера выборки;
    - расчёт длительности A/B-теста.
  3. Мониторинг проведения А/В-теста:
    - проверка распределения пользователей по группам;
    - проверка пересечений пользователей;
    - проверка распределения пользователей по категориальным переменным;
    - вывод после проверки A/B-теста.
  4. Проверка результатов A/B-теста.
  5. Сформулировать вывод по результатам A/B-эксперимента.
 **Обзор метрик, оценка монетизации и юнит-экономики, поиск инсайтов и точек роста, A/B-тестирование маркетплейса**
 — Цель проекта состояла из нескольких частей:
  1. Обзор ключевых метрик, выполнить верхнеуровневую оценку состояния продукта.
  2. Поиск инсайтов, точек роста и сегментация, сформулировать несколько гипотез, которые помогут развить продукт.
  3. Подготовка эксперимента и подведение его результатов:
    - контекст и цель эксперимента;
    - формат эксперимента;
    - проверка распределения пользователей по категориальным переменным;
    - размеры групп и запуск;
    - интерпретация результатов эксперемента.
 4. Выводы и рекомендации по итогам проведенного анализа и проведенного эксперемента.
